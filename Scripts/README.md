# Scripts Directory

Documentation of Excel-based analytical approaches, formulas, and methodologies used in the GameCo sales analysis.

## Contents

- **[Analysis_Methodology.md](Analysis_Methodology.md)**: Complete documentation of:
  - Data cleaning and preparation steps
  - Excel formulas and calculated fields
  - Pivot table configurations
  - Analytical techniques and validation methods
  - Visualization design principles

## Key Techniques Documented

- Regional market share calculations
- Franchise sales aggregation using wildcards
- Year-over-year trend analysis
- Publisher competitive positioning
- Genre preference segmentation

All analysis performed using Microsoft Excel with native functions (SUMIF, pivot tables, charting tools).
